# Federated-Learning-Using-Flower
## Overview 
This project demonstrates a federated learning simulation using the Flower framework and Python.

## Technologies used
- Python
- Flower Framework
- Numpy
- TensorFlow / Pytorch

## Features 
- Client-Server federated architecture
- Distributed model training
- Aggregation of client updates
- Simulation environment

## How to Run
1. Install dependencies
2. Run server.py
3. Run Client.py
